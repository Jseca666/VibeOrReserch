# 推导主线索引

更新时间：2026-04-30

本索引用来把 `01_原始笔记/` 中尚未整理和验证的公式推导重排成可审计的理论链。原始笔记中的展示公式已统一为 `$$...$$`；本索引中的公式使用 `\(...\)` 或 `$$...$$`，避免把数学内容放进代码反引号。

## 公式类型

| 标记 | 含义 | 进入论文前要求 |
|---|---|---|
| 定义 | 变量、映射、集合或风险度量的定义 | 符号表和假设账本一致 |
| 恒等式 | 由代数展开或变量替换得到 | 检查维度、域和值域 |
| 条件命题 | 在额外假设下成立的数学结论 | 命题卡、假设账本、反例搜索、证明审计 |
| 近似 | 线性化、二阶界、网格化、SOC/CVaR 替代 | 说明保守性或误差来源 |
| 优化改革写 | DRCC、MILP、对偶或切平面形式 | 检查凸性、有限维化、强对偶和复杂度 |
| 解释 | 对公式意义的叙述 | 不得单独作为论文强 claim |
| 待验证 | 尚缺证明、文献或数值确认 | 只能作为研究方向或猜想 |

## 总体推导顺序

```text
风预测样本/风变量
  -> 风空间 Wasserstein 模糊集
  -> 能耗映射 Phi 或二维潜在映射 G
  -> 巡检方案能耗 h_x(w)=a(x)^T Phi(w)-B
  -> 决策相关违约集合 V(x)
  -> 样本到违约集合距离 Delta_i(x)
  -> 分布鲁棒机会约束的最坏违约概率
  -> DRCC 有限维化 / 网格 DRCC-MILP / 大规模巡检 MILP
  -> 论文 claim 审批
```

当前最重要的分叉：\(P\) 是否在相关空速区间单射。若单射，可走“可逆流形 Wasserstein 球”版本；若非单调，只能走“折叠流形 / fiber-cost / 二维潜在推前”版本。

当前主线范围锁定：论文主线只关注分布鲁棒机会约束。推前模糊集命题是辅助桥梁，用于解释能耗变量和物理支撑，不作为优化主轴；SRI/CVaR/SOC 只作为背景、对照或后续扩展，不能作为当前主线 claim。

## 第一轮门禁结论

- 可以作为论文主线的安全表述：风致能耗不确定性通过 \(h_x(w)=a(x)^\top\Phi(w)-B\) 进入决策相关违约集合，DRCC 的可解化应围绕样本到坏集距离和最坏违约概率展开。
- 当前最稳的数学核心仍是 C-001；它已经有完整样板，但还缺 measurable selection 或闭包细节。
- 最有潜力的新增理论线是 C-004 -> C-005 -> C-006 的单调可逆版本，以及 C-007 -> C-008 -> C-009 的非单调折叠版本。
- 运筹部分只以分布鲁棒机会约束及离散流形 DRCC-MILP 作为当前可实现主线，DR-CVaR/SOC 暂缓。
- 暂时不应把“等距 Wasserstein 球”“有限 fiber”“严格凸 \(F(u)\)”“MILP 精确连续化”写成已证结论。
