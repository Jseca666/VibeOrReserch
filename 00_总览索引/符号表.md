# 符号表

| 符号 | 含义 | 状态 |
|---|---|---|
| \(\mathcal W\) | 原始风状态空间 | 已用 |
| \(w_i\) | 名义风样本或集合预报成员 | 已用 |
| \(\widehat Q\) | 原始风空间中的离散名义分布 | 已用 |
| \(\Phi\) | 从风状态到能耗/latent-cost 向量的映射 | 已用 |
| \(\Xi\) | 能耗或 latent-cost 空间，通常为 \(\Phi(\mathcal W)\) 或有界盒 | 已用 |
| \(\Phi_{\#}P\) | 分布 \(P\) 在映射 \(\Phi\) 下的推前分布 | 已用 |
| \(W_1,W_2\) | Wasserstein 距离，需明确阶数和地面距离 | 待统一 |
| \(c_i(\xi)\) | 从 \(w_i\) lift 到纤维 \(\Phi^{-1}(\xi)\) 的最小成本 | 已用 |
| \(\mathcal A_\varepsilon\) | 原始风空间 Wasserstein 球推前后的分布集合 | 已用 |
| \(\mathcal B_\varepsilon\) | latent-cost 空间中的等价候选集合 | 已用 |
| \(\mathcal F_\theta\) | 能耗空间 Wasserstein 型模糊集 | 已用 |
| \(\rho_\gamma^E\) | 运营能耗安全风险指数 | 已用 |

## 统一性检查

- `W_1` 和 `W_2` 不能混用；若从 \(W_2^2\) 推导 \(c_i\)，正文必须说明为什么不是 \(W_1\)。
- \(\Xi\) 若取 \(\Phi(\mathcal W)\)，应说明紧性和可测性；若取盒约束，应说明 \(\Phi(\mathcal W)\subseteq \Xi\)。
- \(c_i\) 是否达到最小值依赖 \(\mathcal W\) 紧、\(\Phi\) 连续、纤维闭等条件。
