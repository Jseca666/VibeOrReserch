## 2. 格式裁决

本轮采纳以下分层：

```text
Markdown 摘要 / 审计层
  + CSV / JSONL 机器索引层
  + YAML sidecar 元数据层
  + LaTeX 数学推导层
  + JSON / CSV / log 实验证据层
```

后续 Auto-Gate 新任务默认先读取结构化副本，再按需读取 Markdown、LaTeX、实验日志或原始笔记。Markdown 宽表继续保留人类审计入口，但新增记录优先写结构化副本，并同步 Markdown 摘要。

## 3. 私有边界

- `raw_notes_anchor_map.csv` 和 `raw_notes_theorem_validation_index.jsonl` 含原始笔记片段摘要或候选 statement，默认不推 GitHub。
- `theorem_literature_index.jsonl` 当前为高层文献 grounding；若后续加入未公开推导细节、论文定位细节或原始笔记摘要，应转为私有副本。
