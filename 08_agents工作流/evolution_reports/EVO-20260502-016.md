## 4. 发现并修正的问题

本轮测试发现，旧推导若把距离形式写成存在 \(t\ge0\)，会在 \(\bar\rho=0\) 且存在坏样本时出现假阳性：取 \(t=0\) 会让约束退化为平凡可行。

处理结果：

```text
错误旧写法：t >= 0
本轮修正：t > 0
```

因此，C-011 后续不得使用允许 \(t=0\) 的距离形式。

## 5. 文献边界

本轮使用的代表文献：

- Weijun Xie, *On Distributionally Robust Chance Constrained Programs with Wasserstein Distance*, arXiv:1806.07418。

- \(W_1\) Wasserstein DRCC、样本到 unsafe set 距离、big-M/MIP reformulation 已有强相关文献。

- 可写贡献只能落在风-能耗映射 \(G(z)\)、二维 \(z\) 网格、航向能耗相关性和巡检 MILP 结构化适配上。
