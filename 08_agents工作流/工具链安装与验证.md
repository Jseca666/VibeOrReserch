
```powershell
$env:Path = "$env:USERPROFILE\.elan\bin;$env:Path"
```

## SageMath 安装建议

建议在 WSL + Ubuntu 中配置 SageMath。当前 Ubuntu 发行版尚未安装成功；可在重启 Windows 后再次尝试：

```powershell
wsl --install -d Ubuntu
```

进入 Ubuntu 后，推荐用 conda-forge 安装 SageMath：

```bash
conda create -n sage sage python=3.11 -c conda-forge
conda activate sage
sage --version
```

验证脚本：

```bash
cd /mnt/c/Users/dzw/Desktop/research/风与无人机巡检/06_计算实验
sage finite_pushforward_check.sage
```
