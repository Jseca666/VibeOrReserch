# 假设账本：命题1

| 类别 | 假设 | 当前状态 | 作用 |
|---|---|---|---|
| 空间 | \(\mathcal W\subseteq\mathbb R^d\) 非空紧集 | 建议明确写入 | 保证最小成本达到 |
| 映射 | \(\Phi:\mathcal W\to\mathbb R^q\) 连续 | 建议明确写入 | 保证 \(\Xi=\Phi(\mathcal W)\) 紧，纤维闭 |
| 目标空间 | \(\Xi=\Phi(\mathcal W)\) | 建议明确写入 | 避免空纤维 |
| 概率 | \(\pi_i\ge0,\sum_i\pi_i=1\)，最好 \(\pi_i>0\) | 需说明 | 零权重样本可忽略 |
| Wasserstein | 使用 \(W_2\)，地面成本为欧氏平方距离 | 必须统一 | 与 \(\varepsilon^2\) 和 \(c_i\) 一致 |
| 可测性 | \(c_i\) Borel 可测 | 需证明或引用 | 使 \(\int c_i\,dR_i\) 有意义 |
| 选择 | 存在 measurable minimizer，或使用 \(\eta\)-optimal measurable selector | 关键缺口 | 构造 \(\mu_i\) |
| 闭包 | 若仅有近似选择，需要处理极限闭性 | 关键缺口 | 避免等式变成闭包等式 |

## 推荐定理版本

第一版论文中建议采用紧度量空间和连续映射版本。若要推广到 Polish 空间和 Borel 映射，应单独处理选择定理、下半解析函数和闭包问题。
