# 命题卡：原始风空间推前模糊集与 latent-cost 集合等价

## Claim ID

C-001

## 原始来源

`01_原始笔记/命题1等价推前模糊集.md`

## 命题状态

`条件成立`

当前判断：在紧性、连续性、\(W_2\) 成本一致、以及可测/近似 lift 条件下，等价结论可以成立。最需要补强的是从 \(R_i\) 构造 \(\mu_i\) 时的可测选择或近似选择论证。

## 数学对象

令 \(\mathcal W\subseteq\mathbb R^d\) 为风状态空间，\(\Phi:\mathcal W\to\mathbb R^q\) 为从风状态到能耗或 latent-cost 向量的映射，\(\Xi=\Phi(\mathcal W)\)。名义风分布为

```math
\widehat Q=\sum_{i=1}^N \pi_i\delta_{w_i}.
```

定义

```math
\mathcal A_\varepsilon
=
\{\Phi_{\#}P:\ P\in\mathcal P(\mathcal W),\ W_2(P,\widehat Q)\le\varepsilon\}.
```

对每个 \(i\)，定义纤维 lift 成本

```math
c_i(\xi)=\inf\{|w-w_i|^2:\ w\in\mathcal W,\ \Phi(w)=\xi\}.
```

定义

```math
\mathcal B_\varepsilon
=
\left\{
R=\sum_{i=1}^N\pi_iR_i:\ R_i\in\mathcal P(\Xi),\
\sum_{i=1}^N\pi_i\int_\Xi c_i(\xi)\,dR_i(\xi)\le\varepsilon^2
\right\}.
```

目标结论：

```math
\mathcal A_\varepsilon=\mathcal B_\varepsilon.
```

## 证明路线

1. 使用离散名义分布，把 \(W_2^2(P,\widehat Q)\) 写成从每个 \(w_i\) 出发的条件运输分解。
2. 对任意 \(\mu_i\)，令 \(R_i=\Phi_\#\mu_i\)，利用 \(c_i(\Phi(w))\le |w-w_i|^2\) 证明 \(\mathcal A_\varepsilon\subseteq\mathcal B_\varepsilon\)。
3. 对任意 \(R_i\)，构造或近似构造 \(\mu_i\) 使 \(\Phi_\#\mu_i=R_i\)，并使成本逼近 \(\int c_i\,dR_i\)，证明 \(\mathcal B_\varepsilon\subseteq\mathcal A_\varepsilon\)。

## 风险点

- \(c_i\) 是否可测。
- infimum 是否达到。
- 是否需要 measurable selection。
- 若只可近似达到，最终集合等价是否需要闭包，或是否需要放宽 \(\varepsilon\) 后再取极限。
- 原始笔记中 \(W_1/W_2\) 使用要统一。
