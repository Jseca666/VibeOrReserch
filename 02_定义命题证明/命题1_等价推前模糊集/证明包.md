# 证明包：命题1

## 状态

`PROVABLE AFTER EXTRA ASSUMPTION`

## 目标命题

在如下条件下证明 \(\mathcal A_\varepsilon=\mathcal B_\varepsilon\)：

1. \(\mathcal W\subseteq\mathbb R^d\) 非空紧集。
2. \(\Phi:\mathcal W\to\mathbb R^q\) 连续，\(\Xi=\Phi(\mathcal W)\)。
3. \(\widehat Q=\sum_{i=1}^N\pi_i\delta_{w_i}\)，\(\pi_i\ge0,\sum_i\pi_i=1\)。
4. Wasserstein 距离为 \(W_2\)，地面距离为欧氏距离。
5. 对每个 \(R_i\in\mathcal P(\Xi)\)，允许使用可测最优或 \(\eta\)-最优 lift 选择。

## 引理分解

### 引理 1：离散名义分布的运输分解

对任意 \(P\in\mathcal P(\mathcal W)\)，有

```math
W_2^2(P,\widehat Q)
=
\inf_{\mu_i:\ P=\sum_i\pi_i\mu_i}
\sum_{i=1}^N\pi_i\int_{\mathcal W}|w-w_i|^2\,d\mu_i(w).
```

解释：任意从 \(\widehat Q\) 到 \(P\) 的运输计划可按第一边际原子 \(w_i\) 分解成条件分布 \(\mu_i\)。

### 引理 2：固定 \(R_i\) 的最小 lift 成本

对任意 \(R_i\in\mathcal P(\Xi)\)，

```math
\inf_{\mu_i:\Phi_\#\mu_i=R_i}
\int_{\mathcal W}|w-w_i|^2\,d\mu_i(w)
=
\int_\Xi c_i(\xi)\,dR_i(\xi).
```

其中

```math
c_i(\xi)=\inf_{\Phi(w)=\xi}|w-w_i|^2.
```

该引理是关键技术步。紧性和连续性保证纤维非空紧，最小值存在；可测选择或 \(\eta\)-最优选择保证能从 \(R_i\) 构造 lift。

## 证明草图

### \(\mathcal A_\varepsilon\subseteq\mathcal B_\varepsilon\)

取 \(R\in\mathcal A_\varepsilon\)，则存在 \(P\) 满足 \(R=\Phi_\#P\) 且 \(W_2(P,\widehat Q)\le\varepsilon\)。由引理 1，对任意 \(\eta>0\)，存在 \(\mu_i\) 使

```math
P=\sum_i\pi_i\mu_i,\qquad
\sum_i\pi_i\int |w-w_i|^2\,d\mu_i(w)
\le \varepsilon^2+\eta.
```

令 \(R_i=\Phi_\#\mu_i\)，则 \(R=\sum_i\pi_iR_i\)。又由 \(c_i(\Phi(w))\le |w-w_i|^2\)，有

```math
\sum_i\pi_i\int_\Xi c_i(\xi)\,dR_i(\xi)
\le
\sum_i\pi_i\int_{\mathcal W}|w-w_i|^2\,d\mu_i(w).
```

令 \(\eta\downarrow0\)，得到 \(R\in\mathcal B_\varepsilon\)。

### \(\mathcal B_\varepsilon\subseteq\mathcal A_\varepsilon\)

取 \(R\in\mathcal B_\varepsilon\)，则 \(R=\sum_i\pi_iR_i\) 且

```math
\sum_i\pi_i\int c_i\,dR_i\le\varepsilon^2.
```

由引理 2，对任意 \(\eta>0\)，可取 \(\mu_i\) 使 \(\Phi_\#\mu_i=R_i\) 且

```math
\int |w-w_i|^2\,d\mu_i(w)
\le
\int c_i\,dR_i+\eta.
```

令 \(P=\sum_i\pi_i\mu_i\)，则 \(\Phi_\#P=R\)，且由引理 1，

```math
W_2^2(P,\widehat Q)
\le
\sum_i\pi_i\int |w-w_i|^2\,d\mu_i(w)
\le
\varepsilon^2+\eta.
```

若可取精确最优 lift，则直接得到 \(W_2(P,\widehat Q)\le\varepsilon\)。若只有近似 lift，需要再证明 \(\mathcal A_\varepsilon\) 对弱收敛和二阶矩拓扑闭合，或把结论写成闭包版本。

## 当前最弱步骤

第二个包含方向中，从 \(R_i\) 到 \(\mu_i\) 的 lift 构造。应补充可测选择定理，或将结论改成带闭包/近似半径的版本。
