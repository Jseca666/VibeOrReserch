# 反例搜索记录：命题1

## 目标

在正式接受等价结论前，优先检查该结论在缺少关键假设时是否失败或需要闭包。

## 已检查风险

| 风险 | 检查问题 | 当前结论 |
|---|---|---|
| \(W_1/W_2\) 混用 | 若模糊集用 \(W_1\)，但 \(c_i\) 用平方成本，则公式不匹配 | 必须统一为 \(W_2\) 或重写 \(c_i\) |
| 空纤维 | 若 \(\Xi\) 大于 \(\Phi(\mathcal W)\)，可能存在 \(\Phi^{-1}(\xi)=\varnothing\) | 建议令 \(\Xi=\Phi(\mathcal W)\) |
| 非紧 \(\mathcal W\) | infimum 可能不达到 | 可能需要近似选择和闭包 |
| 不连续 \(\Phi\) | 纤维可能不可闭，\(c_i\) 可测性可能复杂 | 不建议第一版使用 |
| 可测选择缺失 | 从 \(R_i\) lift 回 \(\mathcal W\) 时可能无法选取 minimizer | 是当前主风险 |

## 建议的 Sage/Python 离散检查

离散有限状态版本中，\(\Phi\) 是从有限风点到有限能耗点的映射。此时命题可化为线性规划分解，适合在 `06_计算实验/` 中验证：

1. 枚举有限 \(\mathcal W\) 和映射 \(\Phi\)。
2. 构造离散 \(R_i\)。
3. 解最小 lift 成本。
4. 比较 \(\sum_i\pi_i\int c_i\,dR_i\) 与原始运输成本。

## 当前结论

没有发现紧连续有限维版本下的直接反例；但缺少可测选择/近似选择步骤时，证明仍不应标为完全通过。
