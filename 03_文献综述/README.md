# 文献综述

本目录管理项目文献卡、文献分类索引、文献边界表、BibTeX 和综述草稿。文献工作服务理论命题、优化模型、实验 benchmark、论文新颖性和 claim 审批。

第一版文献层不依赖 API key。优先使用：

- arXiv 检索。
- DOI/论文标题手动检索。
- 用户提供 PDF。
- 网页检索核验来源。
- EvoSkills 辅助检索与综述：`paper-navigator`、`research-survey`、`research-lit`、`novelty-check`。

## 核心文件

- `文献分类总索引.md`：按主题与研究功能双维度管理文献。
- `文献边界表.md`：记录外部已有结论与本项目新增贡献之间的边界。
- `08_agents工作流/产物模板/文献卡模板.md`：单篇文献卡模板。

## 分类主题

- 理论与方法类：`DRO/DRCC理论`、`Wasserstein模糊集`、`机会约束与最坏违约概率`、`MILP reformulation与算法`。
- 应用与背景类：`风不确定性建模`、`风预测与误差校准`、`无人机巡检相关工作`、`无人机路径优化/arc routing`、`无人机能耗/功耗模型`。
- 实验与写作类：`实验benchmark与求解器资料`、`数据集/仿真环境`、`论文写作related work`。

## 文献门禁

- 文献结论不得只停留在对话中；涉及 claim 边界时必须进入文献边界表或研究债务队列。
- 文献边界不清时标注 `文献边界未定`；影响强 claim 时登记为 `claim阻断` 债务。
- 技能输出不能替代 `论文边界与新颖性副导师` 审查或 `PI导师` 裁决。
