# VibeOrReserch

这是“风与无人机巡检”项目的公开评审镜像，仅用于让外部 AI/Pro 审查 agents 工作流、角色分配、自进化闭环、研究债务和 claim 管理机制。

## 公开范围

本镜像包含：

- `AGENTS.md`
- `00_总览索引/` 中的索引、claim、任务和符号状态
- `02_定义命题证明/` 的 README 与假设账本
- `03_文献综述/` 的分类与边界模板
- `04_论文草稿/README.md`
- `07_研究记忆/`
- `08_agents工作流/`

## 未公开范围

本镜像不包含：

- `01_原始笔记/`
- 论文正文 `04_论文草稿/main.tex`
- PDF、LaTeX 编译产物、实验输出、临时 handoff 包
- 私密 token、密钥、环境文件或个人凭据

## 评审目标

请重点审查：

- agents 角色职责是否清晰
- T1-T7 调度是否合理
- 自进化自动工作闭环是否可运行
- 研究债务、角色变更、evolution report、下一轮任务包是否形成闭环
- 当前草稿/claim 状态是否足以进入整理与验证阶段

本镜像不用于证明数学结论，也不用于 claim 升级。
