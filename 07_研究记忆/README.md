# 研究记忆

本目录采用 EvoMemory 思路，但服务于当前数学/运筹项目。

## 文件

- `方向记忆.md`：记录可行/失败研究方向。
- `证明策略记忆.md`：记录可复用证明套路、失败证明路线和常见缺口。
- `实验记忆.md`：记录 Sage/Lean/数值实验经验。

## 写回边界

- 研究记忆只记录跨周期稳定经验，不把未审查 claim、猜想或一次性实验异常写成事实。
- 研究记忆不关闭研究债务，不替代 `研究债务队列.md`、claim 审批表、文献边界表、Auto-Gate 门禁裁决或最终对外责任确认。
- 每轮是否写回研究记忆，应先在 `自进化自动工作闭环协议.md` 的 evolution report 中说明。
