# 方向记忆

本文件承担 EvoScientist 风格 \(M_I\) 的轻量角色：记录可行方向、暂缓方向、失败方向和需谨慎方向。方向记忆不替代 claim 审批、研究债务关闭、文献边界审查或 `PI导师` 裁决。

## 字段说明

| 字段 | 含义 |
|---|---|
| 方向 | 可复用研究方向或需要避开的方向 |
| 记忆类型 | 可行方向 / 待门禁方向 / 失败方向 / 仅作警示 |
| 证据来源 | EVO-ID / RD-ID / NXT-ID / 命题卡 / 文献边界 |
| 状态边界 | 稳定经验 / 待门禁 / claim阻断 / 非当前主线 |

## 写回规则

- IDE 触发时，只能把经过 EVO/RD/NXT 证据支持的方向写入本文件。
- 若方向仍受 `claim阻断` 债务阻断，状态边界必须写明 `claim阻断` 或 `待门禁`。
- 未经文献边界审查的新颖性判断、未审查 claim 和一次性直觉不得写成稳定方向。
