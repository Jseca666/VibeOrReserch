可以。我们把它证明清楚。

为了避免技术细节干扰，先假设：

[
\mathcal W\subseteq \mathbb R^d
]

是紧集，(\Phi:\mathcal W\to \mathbb R^q) 连续。于是

[
\Xi=\Phi(\mathcal W)
]

也是紧集。名义风分布为

[
\widehat Q=\sum_{i=1}^N \pi_i\delta_{w_i}.
]

我们要证明下面两个集合相等。

原始风空间推前集合：

# [ \mathcal A_\varepsilon

\left{
\Phi_{#}P:
P\in\mathcal P(\mathcal W),
W_2(P,\widehat Q)\le \varepsilon
\right}.
]

能耗空间 latent-cost 集合：

# [ \mathcal B_\varepsilon

\left{
R=\sum_{i=1}^N\pi_iR_i:
R_i\in\mathcal P(\Xi),
\sum_{i=1}^N\pi_i\int_{\Xi}c_i(\xi),dR_i(\xi)\le \varepsilon^2
\right},
]

其中

# [ c_i(\xi)

\inf
\left{
|w-w_i|^2:
w\in\mathcal W,\ \Phi(w)=\xi
\right}.
]

要证明：

[
\boxed{
\mathcal A_\varepsilon=\mathcal B_\varepsilon.
}
]

------

## 1. 第一步：把风 Wasserstein 距离拆成从每个 (w_i) 出发的扰动

因为

[
\widehat Q=\sum_{i=1}^N\pi_i\delta_{w_i}
]

是离散分布，所以从 (\widehat Q) 搬运到某个真实风分布 (P) 时，可以理解成：

第 (i) 堆质量 (\pi_i) 原来在 (w_i)，现在被搬成某个条件分布 (\mu_i)。

于是

[
P=\sum_{i=1}^N\pi_i\mu_i.
]

搬运成本就是

[
\sum_{i=1}^N
\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

因此有等式：

# [ \boxed{ W_2^2(P,\widehat Q)

\inf_{\mu_i:\ P=\sum_i\pi_i\mu_i}
\sum_{i=1}^N
\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
}
]

这个等式的意思是：从离散名义分布 (\widehat Q) 到 (P) 的最优运输，可以分解为从每个原子点 (w_i) 出发的条件运输。

所以原始集合 (\mathcal A_\varepsilon) 等价于：

# [ \mathcal A_\varepsilon

\left{
\sum_{i=1}^N\pi_i\Phi_{#}\mu_i:
\mu_i\in\mathcal P(\mathcal W),
\sum_{i=1}^N
\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w)
\le \varepsilon^2
\right}.
]

这一步还没有引入 (c_i(\xi))。

------

## 2. 第二步：证明 (c_i(\xi)) 正好是能耗空间里的最小 lift 成本

固定一个 (i)。给定一个能耗分布 (R_i\in\mathcal P(\Xi))，我们问：

有哪些风分布 (\mu_i) 可以生成它？

条件是

[
\Phi_{#}\mu_i=R_i.
]

也就是说，如果

[
W\sim \mu_i,
]

那么

[
\Phi(W)\sim R_i.
]

在所有这样的风分布 (\mu_i) 里面，最小运输成本是多少？

我们要证明：

# [ \boxed{ \inf_{\mu_i:\ \Phi_{#}\mu_i=R_i} \int_{\mathcal W} |w-w_i|^2,d\mu_i(w)

\int_{\Xi}
c_i(\xi),dR_i(\xi).
}
]

这个式子是整个等价证明的关键。

------

### 2.1 先证 “(\ge)” 方向

任取一个满足

[
\Phi_{#}\mu_i=R_i
]

的风分布 (\mu_i)。

对任意 (w\in\mathcal W)，令

[
\xi=\Phi(w).
]

因为 (w) 本身就是一个可以生成 (\xi) 的风状态，所以

# [ c_i(\xi)

\inf_{\Phi(u)=\xi}
|u-w_i|^2
\le
|w-w_i|^2.
]

也就是

[
c_i(\Phi(w))
\le
|w-w_i|^2.
]

对 (\mu_i) 积分：

[
\int_{\mathcal W}
c_i(\Phi(w)),d\mu_i(w)
\le
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

由于

[
\Phi_{#}\mu_i=R_i,
]

所以

# [ \int_{\mathcal W} c_i(\Phi(w)),d\mu_i(w)

\int_{\Xi}
c_i(\xi),dR_i(\xi).
]

因此

[
\int_{\Xi}
c_i(\xi),dR_i(\xi)
\le
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

因为这个不等式对所有 lift (\mu_i) 都成立，所以

[
\int_{\Xi}
c_i(\xi),dR_i(\xi)
\le
\inf_{\mu_i:\ \Phi_{#}\mu_i=R_i}
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

即

[
\boxed{
\inf_{\mu_i:\ \Phi_{#}\mu_i=R_i}
\int
|w-w_i|^2,d\mu_i(w)
\ge
\int
c_i(\xi),dR_i(\xi).
}
]

------

### 2.2 再证 “(\le)” 方向

现在要证明存在一个最优 lift，使得成本恰好等于

[
\int c_i(\xi),dR_i(\xi).
]

对每个能耗向量 (\xi\in\Xi)，定义它的 fiber：

# [ \Phi^{-1}(\xi)

{w\in\mathcal W:\Phi(w)=\xi}.
]

因为 (\xi\in\Xi=\Phi(\mathcal W))，所以这个集合非空。

又因为 (\mathcal W) 紧、(\Phi) 连续，所以

[
\Phi^{-1}(\xi)
]

是非空紧集。

函数

[
w\mapsto |w-w_i|^2
]

连续，因此在 fiber 上可以取到最小值。也就是说，对每个 (\xi)，存在一个风状态 (s_i(\xi))，使得

[
\Phi(s_i(\xi))=\xi
]

并且

[
|s_i(\xi)-w_i|^2=c_i(\xi).
]

直观地说：

# [ s_i(\xi)

\text{所有能生成 }\xi\text{ 的风里面，离 }w_i\text{ 最近的那个。}
]

现在用这个选择函数 (s_i) 把能耗分布 (R_i) lift 回风空间：

[
\mu_i=(s_i)_{#}R_i.
]

也就是说，先抽一个能耗向量

[
\xi\sim R_i,
]

然后把它映射回

[
w=s_i(\xi).
]

于是

[
w\sim \mu_i.
]

因为

[
\Phi(s_i(\xi))=\xi,
]

所以

[
\Phi_{#}\mu_i=R_i.
]

也就是说，这个 (\mu_i) 确实可以生成 (R_i)。

它的成本为

# [ \int_{\mathcal W} |w-w_i|^2,d\mu_i(w)

\int_{\Xi}
|s_i(\xi)-w_i|^2,dR_i(\xi).
]

由 (s_i) 的定义，

[
|s_i(\xi)-w_i|^2=c_i(\xi).
]

所以

# [ \int_{\mathcal W} |w-w_i|^2,d\mu_i(w)

\int_{\Xi}
c_i(\xi),dR_i(\xi).
]

因此

[
\boxed{
\inf_{\mu_i:\ \Phi_{#}\mu_i=R_i}
\int
|w-w_i|^2,d\mu_i(w)
\le
\int
c_i(\xi),dR_i(\xi).
}
]

和前面的 “(\ge)” 合起来：

# [ \boxed{ \inf_{\mu_i:\ \Phi_{#}\mu_i=R_i} \int_{\mathcal W} |w-w_i|^2,d\mu_i(w)

\int_{\Xi}
c_i(\xi),dR_i(\xi).
}
]

这一步说明：

[
\boxed{
c_i(\xi)
\text{ 精确记录了从 }w_i\text{ 生成能耗点 }\xi\text{ 所需的最小风空间成本。}
}
]

------

## 3. 证明 (\mathcal A_\varepsilon\subseteq \mathcal B_\varepsilon)

现在取任意

[
R\in\mathcal A_\varepsilon.
]

根据定义，存在某个风分布 (P)，使得

[
R=\Phi_{#}P,
]

并且

[
W_2(P,\widehat Q)\le \varepsilon.
]

由第一步的分解，存在 (\mu_1,\ldots,\mu_N)，使得

[
P=\sum_{i=1}^N\pi_i\mu_i,
]

且

[
\sum_{i=1}^N
\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w)
\le
\varepsilon^2.
]

定义

[
R_i=\Phi_{#}\mu_i.
]

那么

# [ R=\Phi_{#}P

# \Phi_{#} \left( \sum_i\pi_i\mu_i \right)

# \sum_i\pi_i\Phi_{#}\mu_i

\sum_i\pi_iR_i.
]

而且

[
\int_{\Xi}
c_i(\xi),dR_i(\xi)
\le
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

所以

[
\sum_i\pi_i
\int_{\Xi}
c_i(\xi),dR_i(\xi)
\le
\sum_i\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w)
\le
\varepsilon^2.
]

因此

[
R\in \mathcal B_\varepsilon.
]

所以

[
\boxed{
\mathcal A_\varepsilon\subseteq \mathcal B_\varepsilon.
}
]

------

## 4. 证明 (\mathcal B_\varepsilon\subseteq \mathcal A_\varepsilon)

现在反过来，取任意

[
R\in\mathcal B_\varepsilon.
]

根据定义，存在

[
R_1,\ldots,R_N\in\mathcal P(\Xi),
]

使得

[
R=\sum_{i=1}^N\pi_iR_i,
]

并且

[
\sum_i\pi_i
\int_\Xi c_i(\xi),dR_i(\xi)
\le
\varepsilon^2.
]

由第二步的结论，对每个 (R_i)，存在风分布 (\mu_i)，满足

[
\Phi_{#}\mu_i=R_i,
]

并且

# [ \int_{\mathcal W} |w-w_i|^2,d\mu_i(w)

\int_{\Xi}
c_i(\xi),dR_i(\xi).
]

现在定义一个整体风分布

[
P=\sum_{i=1}^N\pi_i\mu_i.
]

它通过 (\Phi) 推到能耗空间后是

# [ \Phi_{#}P

# \sum_i\pi_i\Phi_{#}\mu_i

# \sum_i\pi_iR_i

R.
]

再看它离 (\widehat Q) 的 Wasserstein 距离。由第一步的运输分解，

[
W_2^2(P,\widehat Q)
\le
\sum_i\pi_i
\int_{\mathcal W}
|w-w_i|^2,d\mu_i(w).
]

代入上面的等式：

[
W_2^2(P,\widehat Q)
\le
\sum_i\pi_i
\int_{\Xi}
c_i(\xi),dR_i(\xi)
\le
\varepsilon^2.
]

所以

[
W_2(P,\widehat Q)\le \varepsilon.
]

并且

[
R=\Phi_{#}P.
]

因此

[
R\in\mathcal A_\varepsilon.
]

所以

[
\boxed{
\mathcal B_\varepsilon\subseteq \mathcal A_\varepsilon.
}
]

------

## 5. 合并两边

我们已经证明了

[
\mathcal A_\varepsilon\subseteq \mathcal B_\varepsilon
]

和

[
\mathcal B_\varepsilon\subseteq \mathcal A_\varepsilon.
]

因此

[
\boxed{
\mathcal A_\varepsilon=\mathcal B_\varepsilon.
}
]

也就是

# [ \boxed{ \left{ \Phi_{#}P: W_2(P,\widehat Q)\le \varepsilon \right}

\left{
R=\sum_i\pi_iR_i:
\sum_i\pi_i\int c_i(\xi),dR_i(\xi)\le \varepsilon^2
\right}.
}
]

------

## 6. 这个证明的核心直觉

原始风空间模型是：

[
w_i
\longrightarrow
w
\longrightarrow
\xi=\Phi(w).
]

能耗空间模型把中间的 (w) 隐藏掉了，直接写成：

[
w_i
\longrightarrow
\xi.
]

但为了不丢失信息，它给每个 (\xi) 配了一个成本：

# [ c_i(\xi)

\min_{\Phi(w)=\xi}
|w-w_i|^2.
]

所以 (c_i(\xi)) 的作用就是：

[
\boxed{
\text{虽然我们只看见能耗向量 }\xi，
\text{但仍然记得它背后至少需要怎样的风扰动才能产生。}
}
]

因此，这个能耗空间表达不是松弛，而是精确等价。

------

## 7. 一个小技术提醒

上面证明里我用了一个选择函数

[
s_i(\xi)\in\arg\min_{\Phi(w)=\xi}|w-w_i|^2.
]

这个函数存在的充分条件是：

[
\mathcal W \text{ 紧},\qquad \Phi \text{ 连续}.
]

在你的问题里，(\mathcal W) 通常由物理风速边界、ensemble 凸包或盒约束给出，所以把它设成紧集是自然的。

如果 (\mathcal W) 不紧，或者 (\Phi) 不连续，那么严格等价可能要写成闭包形式，或者用近似选择：

[
|s_i^\delta(\xi)-w_i|^2
\le
c_i(\xi)+\delta.
]

这时会得到

[
\mathcal B_\varepsilon
\subseteq
\overline{\mathcal A_\varepsilon}
]

或者

[
\mathcal B_\varepsilon
\subseteq
\mathcal A_{\varepsilon+\eta}.
]

但在紧支撑、连续功耗映射下，前面的等价是严格成立的。
