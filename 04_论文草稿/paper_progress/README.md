# Paper Progress

本目录存放成果型论文 LaTeX 进度片段。它只承接已经通过文献边界、证明审计、反例测试或实验门禁，并经 Auto-Gate 输出准入的内容，不直接替代 `main.tex`。

## 使用规则

- 每个进度片段应有 `PG-YYYYMMDD-序号` 或清晰主题名。
- 每个片段必须标注来源 OUT/NXT/EVO/RD、claim 状态、准入证据和允许用途。
- 未通过门禁的内容不得进入本目录，应留在 `technical_notes/`、proof package、RD/NXT/EVO 或实验日志中继续推进。
- 不得把 `仅有推导`、`待验证`、`系统内候选` 或 `被 RD 阻断` 内容写成论文进度片段。

## 默认状态

当前 `main.tex` 仍视为样板稿。正式论文正文更新前，先在本目录形成成果型、可审计、可追溯的论文片段。
