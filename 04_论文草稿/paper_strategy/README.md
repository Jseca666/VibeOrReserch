# Paper Strategy 目录说明

本目录是论文方向与贡献策略层，不是论文正文、不是 slides，也不替代 claim 审批、研究债务、证明审计或 Auto-Gate 裁决。

## 定位

`paper_strategy/` 用于回答：

- 这篇论文要解决什么研究问题。
- 当前候选贡献是什么。
- 哪些文献构成 related work 主线。
- 哪些技术产物、证明、实验和文献边界支撑贡献。
- 哪些内容只能作为内部技术债务、附录候选或暂缓方向。

## 使用规则

- 重技术 NXT 启动前，应先检查是否能绑定本目录中的论文主题、贡献候选或证据缺口。
- technical note、proof package、paper progress 和 slides 若无法追溯到本目录，应由 Auto-Gate 暂缓成果型输出。
- 本目录可以记录候选贡献和叙事假设，但不得把未验证 claim 写成已成立结论。
